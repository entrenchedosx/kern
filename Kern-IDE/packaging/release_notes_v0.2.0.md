## Desktop (Tk) — `desktop-tk/`

- **Versioning:** `VERSION` file (0.2.0); shown in window title, toolbar, and **Help → About**.
- **PyInstaller:** `kern-ide.spec` bundles `VERSION` for frozen builds.
- **Features:** command palette (Ctrl+Shift+P), Problems list (double-click to jump), first-run quick start, explorer right-click (new/rename/delete), preferences (font + autosave), breadcrumbs, workspace picker.

Matches sync from [kern](https://github.com/entrenchedosx/kern) tree `Kern-IDE/` at this tag.

**Note:** Install the **portable `.zip` asset** on this release (or newer) for a single-file `kern-ide.exe` build. Source-only assets do not include the executable.
