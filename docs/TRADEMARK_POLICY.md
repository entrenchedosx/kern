# Trademark and Naming Policy

The code is open source, but project naming/branding should not mislead users.

## Allowed

- "Fork of Kern"
- "Built with Kern"
- "Kern-compatible"

## Not allowed without permission

- Presenting a fork as the official Kern project.
- Using "Kern" name/logo in a way that implies official endorsement.
- Releasing a product under the exact same branding while changing ownership claims.

## Recommendation for forks

- Use a distinct project name.
- Keep a visible note that the project derives from Kern, with upstream link.
