# Attribution Policy

This project is open source under GPLv3, but attribution clarity is expected in public use.

## Expected attribution for public forks/derivatives

- Keep upstream project credit visible in README/docs:
  - "Based on Kern" (or equivalent wording)
  - link to upstream repository: `https://github.com/entrenchedosx/kern`
- Preserve existing copyright/license notices in source files.
- Do not remove author credit from copied sections.

## Why this exists

Kern is a solo-built project, and attribution helps users find the upstream source,
track original work, and understand lineage across forks.

## Scope

This document describes community expectations and does not replace the legal text
in [LICENSE](../LICENSE).
