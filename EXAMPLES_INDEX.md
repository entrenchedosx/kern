## 🚀 Running Examples

### Basic Examples
```bash
./kern examples/language/01_variables_and_types.kn
```

### Graphics Examples (requires g3d)
```bash
./kern examples/graphics/clean_ursina_test.kn
./kern examples/games/snake.kn
```

### With Build
```bash
cd build
./kern ../examples/language/04_functions.kn
```
