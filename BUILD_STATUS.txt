================================================================================
HOW TO BUILD (Windows with Visual Studio)
================================================================================

METHOD 1: Using Batch Script (Recommended)
-------------------------------------------
1. Open: Start Menu -> Visual Studio 2022 -> 
         "x64 Native Tools Command Prompt for VS 2022"

2. Navigate to project:
   cd d:\simple_programming_language

3. Run build script:
   BUILD_3DENGINE.bat

4. Wait for compilation (2-5 minutes)

METHOD 2: Using PowerShell
---------------------------
1. Open PowerShell as Administrator

2. Navigate to project:
   cd d:\simple_programming_language

3. Run build script:
   .\BUILD_3DENGINE.ps1

4. Wait for compilation (2-5 minutes)

METHOD 3: Manual CMake
---------------------
1. Open "x64 Native Tools Command Prompt"

2. Create build directory:
   mkdir build-3dengine
   cd build-3dengine

3. Configure:
   cmake .. -G "Visual Studio 17 2022" -A x64 -DCMAKE_BUILD_TYPE=Release

4. Build:
   cmake --build . --config Release --target kern --parallel 4

================================================================================
TROUBLESHOOTING
================================================================================

ERROR: "Visual Studio environment not detected!"
  → You must run the batch script from "x64 Native Tools Command Prompt"
  → NOT from regular Command Prompt or PowerShell

ERROR: "CMake configuration failed"
  → Ensure CMake is installed: cmake --version
  → If not installed: winget install Kitware.CMake

ERROR: "LINK : fatal error LNK1181: cannot open input file"
  → Clean build: Delete build-3dengine directory and rebuild

ERROR: "3dengine.cpp(1): fatal error C1083: Cannot open include file"
  → Ensure all source files exist in kern/modules/3dengine/
  → Check that #include paths are correct
